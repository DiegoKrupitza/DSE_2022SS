# DSE_2022SS

## Development

### Installation

```bash
yarn install && yarn install:all
```

### Docker

#### Start

```bash
yarn dev
````

#### Stop

```bash
yarn down
```

### Kubernetes

Local Kubernetes development requires [kompose](https://kompose.io) and [kubectl](https://kubernetes.io/docs/tasks/tools/#kubectl).

#### Generate files

```bash
yarn kompose
```

#### Start Kubernetes

```bash
yarn kubernetes:up
```

#### Stop Kubernetes

```bash
yarn kubernetes:down
```
